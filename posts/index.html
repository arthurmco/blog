Não dá pra usar git-lfs no blog

2022-07-29 :: Arthur Mendes
#short  #meta 

Se você entrou no blog e percebeu que as imagens sumiram, é porque eu estava experimentando colocar as imagens no git-lfs

O git-lfs é uma extensão do git que guarda certos arquivos, geralmente binários, separadamente, em um servidor separado, o que é bom, já que o git não é otimizado pra arquivos binários.

Pra quem estiver afim de usar, já saiba que não rola.
Ironicamente, essa extensão tem dedo da própria equipe do GitHub.

Felizmente, agora as coisas já tevem ter normalizado. Desculpem o transtorno.
